# Breast Cancer Segmentation using CAE and U-Net

This project compares the performance of Convolutional Autoencoder (CAE) and U-Net for breast cancer segmentation in mammogram images using the CBIS-DDSM dataset.

## Framework

- Python
- TensorFlow
- Streamlit

## Models

- Convolutional Autoencoder (CAE)
- U-Net

## Dataset

CBIS-DDSM
